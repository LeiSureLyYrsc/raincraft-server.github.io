为了玩家账户安全，照顾非正版玩家体验，RainCraft采用了 [**自建皮肤站**](https://skin.ineko.cc/) 供非正版玩家们提供了可以进入服务器的另一种登录方式

如果你没有正版账号(或是离线玩家)，请你点击[**为你的客户端应用外置登录**](/docs/authlib-client/)进行操作。
